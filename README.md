**Curology**
---
Curology is a frontend project that I personally designed to replicate a modern skincare website experience, originally designed first in Figma and then developed using HTML, CSS, and JavaScript. This project served as a hands-on opportunity for me to explore Figma’s design workflow, which eventually helped me to have a better understanding of the aspects of UI/UX. I then coded the entire website using vanilla HTML/CSS/JavaScript, but mainly prioritized the design and aesthetic of the website. Through this project, I managed to gain practical experience in layout structuring, styling, and interactive behavior in a real-world-inspired application.

All assets used that I have used in this project were sourced from Unsplash, which provides free-to-use images under its license. Attribution is not required but is appreciated.
